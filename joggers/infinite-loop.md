# Infinite Loop Joggers

Minimal, ultra-soft joggers built for all-day deep work and quick coffee runs.

## Features
- **Fabric**: Modal (92%) + Elastane (8%) 320–340 gsm — 4-way stretch, breathable, drapes clean
- **Fit**: Relaxed top block, tapered leg; rib cuff prevents chair-wheel snag
- **Pockets**: Two hand pockets + one **zip** media pocket (silent zips)
- **Waist**: Wide elastic with flat drawcord; internal key loop
- **Branding**: Tiny tone-on-tone `deep@work` hit; optional thigh print `infinite loop`

## Colors
- Dark Mode Black / Slate Grey / Oatmeal

## Sizes
XS–XXL (unisex). Inseam reference (M): 74 cm.

## Care
Machine wash cold, gentle; tumble low or hang; cool iron (avoid prints).

## Add-Ons
- Patch packs (syntax colors)
- Event heat-press customization
